# Croc2QR

A simple ruby script to generate QR code from croc code.


Usage: 

## Send a file
```
./croc2qr <filepath>
```
And a qr.svg will be generated at the same location as the filepath

## Generate random qr code for croc code

```
./croc2qr -g
```


## receive using the last generated code

```
./croc2qr -f
```
